# Instrucciones para Claude Code

Lee primero `AGENTS.md` y sigue su jerarquía de instrucciones.

Antes de editar, inspecciona Git con `git status --short`, confirma la rama actual y registra el commit base. No sobrescribas cambios existentes sin commit ni cambios del usuario.

Al terminar, entrega un resumen de archivos modificados, comandos ejecutados y validaciones realizadas. Los reportes de tareas importantes se guardan en `docs/reports/`.

No hagas commit ni push sin autorización explícita.
