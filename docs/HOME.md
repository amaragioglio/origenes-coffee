# Orígenes Coffee - Home

Índice principal para Obsidian y agentes.

## Estado actual

Proyecto estático de Orígenes Coffee con portada editorial, páginas por país, formulario de leads con Formspree pendiente de ID real, atribución UTM first-touch en `sessionStorage`, `analytics-config.js` con IDs vacíos y Vercel Web Analytics cargado en las páginas.

## Próxima tarea

Auditar el formulario actual y decidir si se mantiene Formspree o se diseña una migración a Supabase.

## Decisiones pendientes

- Definir modelo final de datos.
- Definir consentimiento y política de privacidad.
- Definir IDs de tracking.
- Definir criterios de éxito del smoke test.
- Definir presupuesto y duración de campañas.
