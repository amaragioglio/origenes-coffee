# Archivo

Esta carpeta guarda documentos reemplazados, decisiones obsoletas, reportes antiguos que ya no son operativos y materiales históricos que conviene conservar.

Mover un documento aquí cuando haya sido sustituido por una versión vigente o cuando ya no sea una instrucción actual.

Los archivos archivados no son instrucciones actuales. Deben conservar fecha, motivo de archivo y referencia al documento que los reemplaza cuando exista.
