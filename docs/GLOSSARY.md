# Glosario

- Smoke test: prueba pequeña para validar interés real antes de invertir más.
- Lead: contacto capturado, normalmente email, con intención potencial.
- Landing page: página diseñada para recibir tráfico de campaña y generar una acción.
- Landing country: país de la página de aterrizaje visitada.
- Identity country: país con el que la persona se identifica cultural o familiarmente.
- Preferred coffee origin: origen de café preferido declarado o inferido.
- Conversion rate: porcentaje de visitantes que completan una acción.
- Cost per lead: gasto de campaña dividido por leads atribuidos.
- First-touch attribution: atribución al primer contacto registrado.
- Last-touch attribution: atribución al último contacto antes de convertir.
- UTM: parámetros de URL para identificar fuente, medio, campaña, contenido y término.
- ADR: Architecture Decision Record; registro de una decisión técnica o de producto.
- RLS: Row Level Security; reglas de acceso por fila en bases de datos como Supabase.
- PII: información personal identificable.
- CTA: call to action; llamada a la acción.
- GA4: Google Analytics 4.
- Clarity: Microsoft Clarity, herramienta de analítica de comportamiento.
