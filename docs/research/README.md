# Investigación

Esta carpeta guarda investigación de mercado, referencias de diseño, análisis de campañas, notas de entrevistas y fuentes externas útiles para decisiones del proyecto.

Cada documento debe incluir fecha, metodología, fuentes y alcance. Las fuentes deben citarse con URL, autor u organización cuando exista, fecha de consulta y una nota breve sobre por qué se usa.

Separar claramente:

- Hechos confirmados.
- Hipótesis.
- Interpretaciones.

La investigación original existente está en [[../../INVESTIGACION]].

No guardar datos personales de participantes en esta carpeta.
